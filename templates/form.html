Instructions in Creating Test Items or Questions
  • Section: You can write several test sections, and at least one section should be present. Every test section should be preceeded with # and followed with the section name.
    Example:
      # Test 1
  • Question: You can write several questions in a single section. Every question should be preceeded with * .
    Example:
      * What is the color of the sky?
  • Answer option: Every questions should has at least one correct answer option. An options should be preceeded with 4 spaces and - . The correct answer option should has (correct) or (answer) at the end of the line. You can have more than one correct answer in a single question
    Correct answer option:
          - blue (correct)
    Wrong answer option:
          - red
  • Image: You can embed the image by writing this format: ![image caption](image url)
    Example:
      ![logo](https://website.com/logo.png)
  • Math Formula: You can embed math formula in this format: $$latex-formula$$
    Example:
      $$\alpha + \beta$$
