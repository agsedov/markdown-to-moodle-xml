Markdown to Moodle


  • Section: You can write several test section and at least one section should be present. Every test section should be preceeded with # and followed with the section name.
    Example:
      # Test 1
  • Question: You can write several questions in a single section. Every question should be preceeded with * .
    Example:
      * What is the color of the sky?
  • Answer option: Every questions should has at least one correct answer option. An options should be preceeded with - , and the correct answer option should has a single space at the end of the line. You can have more than one correct answer in a single question, and the weight of each answer will be calculated automatically
    Correct answer option:
          - blue
    Wrong answer option:
          - red
  • Image: You can embed the image by writing this format: ![image caption](image url)
    Example:   ![logo](https://website.com/logo.png)
  • Math Formula: You can embed math formula in this format: $$latex-formula$$
    Example:
      $$\alpha + \beta$$
